-專案名稱與目的： 連接資料庫的 JavaFX 記帳軟體專案，以幫助政大學生直覺管理個人帳務為旨

-角色貢獻：

1. 負責資料流程設計與系統整合，串接 JavaFX 與 MySQL，建構使用者登入、資料紀錄與更新機制

2. 開發支出圓餅圖模組，實作即時更新與視覺優化，提升資料呈現的清晰度與可讀性

3. 協助將使用需求轉化為具體功能設計，確保系統操作符合使用情境

-技術： JavaFX、MySQL、資料庫基本操作、資料視覺化（PieChart）

-專案成果：

1. 建立完整從資料輸入、儲存到視覺化呈現的應用流程

2. 提升使用者對支出結構的理解，強化記帳行為的持續性

3. 在課程專案中獲得教授對於系統設計與視覺呈現的正面回饋
